Appairer un émetteur

  1. Reportez-vous à la documentation de l'émetteur pour lui faire émettre un signal de reconnaissance.
    Dans le cas des émetteurs de type PTM (interrupteurs, supports de badge, poignées mécaniques, etc.), actionnez-les (appuyer sur l'interrupteur, insérer un badge, bouger la poingée) pour provoquer l'envoi d'un signal.
  2. Lorsque PlaceTouch reçoit le signal transmis par l'émetteur, une vignette correspondant à cet émetteur apparaît dans l'espace "Dispositifs découverts".
  3. Dans l'espace "Dispositifs découverts", cliquez sur la vignette correspondante à l'émetteur que vous souhaitez ajouter à PlaceTouch pour ouvrir un panneau de caractéristiques.
  4. Précisez un nom et une localisation pour l'émetteur, en complétant le panneau de caractéristiques dans la colonne de droite.
  5. Cliquez sur "Ajouter le dispositif" pour terminer l'appairage de l'émetteur avec le boitier PlaceTouch.

Ajouter un récepteur

Utilisez le panneau "Ajout manuel" ci-dessous.
  1. Précisez un nom pour votre nouveau récepteur.
  2. Précisez sa localisation.
  3. Sélectionnez son profil.
  4. Cliquez sur "Ajouter" pour ajouter le dispositif au boitier PlaceTouch.
